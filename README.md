# 🌱 QuantSeed 量化种子

**Phase 1 - 离线回测与策略监控平台**

一个基于 Python 的本地量化交易平台，支持自动化数据管理、事件驱动回测、可视化监控仪表盘。

---

## 功能特性

- **自动化数据管理**：通过 akshare 免费获取 A 股历史日线数据（前复权），本地 CSV 存储，支持增量更新
- **策略回测引擎**：基于 backtrader 的事件驱动回测系统，内置双均线交叉策略（MA20 × MA60）
- **可视化仪表盘**：Streamlit 构建的 Web 界面，展示绩效指标、资金曲线、交易明细
- **信号生成**：独立于回测框架，基于最新行情自动生成次日预选买入信号清单

## 快速开始

### 1. 环境要求

- Python 3.9+
- Windows / macOS / Linux

### 2. 安装依赖

```bash
cd QuantSeed
pip install -r requirements.txt
```

### 3. 启动平台

```bash
streamlit run src/app.py
```

浏览器将自动打开 `http://localhost:8501`。

### 4. 使用流程

1. **更新数据**：在左侧边栏点击「更新数据」按钮，首次运行会下载全部股票自 2018 年至今的日线数据
2. **设置回测参数**：选择回测日期、股票和初始资金
3. **开始回测**：点击「开始回测」，在仪表盘查看绩效
4. **生成信号**：点击「生成今日信号」获取最新的金叉股票清单

## 策略说明

### 双均线交叉策略（MA20 × MA60）

| 项目 | 说明 |
|------|------|
| 买入信号 | MA20 上穿 MA60（金叉），且当前无持仓 |
| 卖出信号 | MA20 下穿 MA60（死叉），且当前有持仓 |
| 仓位管理 | 每次全仓买入（95%仓位，留5%现金） |
| 成交模拟 | 信号次日开盘价成交（模拟T+1） |
| 手续费 | 万2.5（可配置） |
| 滑点 | 千分之一（可配置） |

## 配置说明

所有可调参数集中在 `src/config.py` 中：

```python
MA_FAST = 20           # 快线周期
MA_SLOW = 60           # 慢线周期
POSITION_SIZE = 0.95   # 仓位比例
INITIAL_CASH = 1000000 # 初始资金
COMMISSION_RATE = 0.00025  # 手续费率
```
